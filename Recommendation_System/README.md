 Recommendation System 

This project is part of my CodSoft AI Internship.

It is a content-based recommendation system that suggests items like movies, books, and music based on user preferences.

## Features
- GUI-based system
- Multi-domain recommendations
- Content-based filtering
- Simple and user-friendly interface

## How it Works
The system recommends items based on selected type and genre using content-based filtering.
