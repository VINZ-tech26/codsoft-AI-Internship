#  AI Chatbot – Task 2

This project is developed as part of my AI Internship at CodSoft.

It is a Python-based chatbot with a GUI interface that can interact with users and remember simple information using JSON memory.

## Features
- GUI chatbot built with Tkinter
- Stores user information (name, likes)
- Retrieves stored information when asked
- Rule-based chatbot responses

##  Technologies
- Python
- Tkinter
- JSON

## Run

python chatbot_gui.py
